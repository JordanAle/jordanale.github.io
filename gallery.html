Design

Prototype for a tool to help students create balance between digital and print reading habits. User research indicated that students valued digital reading, but wanted to use additional print media. My design holds both mediums as valuable and uses social motivation, action facilitation and personal reward setting to help achieve balance. Click the thumbnail to explore the designs in Figma.

Journey Map and System Design for a proposed Citizen Journal application centering specific vulnerable communities. Responding to social media platform changes that began circa 2018 and increased distrust between users and platforms, this application has democratic user control at its core. Some tools that allow this are (i) special voting and suggestion systems for community standards and (ii) a bottom-up community moderation system and (iii) accessibilty across several device types.
